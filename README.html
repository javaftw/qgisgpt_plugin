QGISGPT Plugin

The companion plugin for the (unofficial) multi-purpose AI assistant for QGIS users.

Prerequisites

You will need an OpenAI API key.

  1. Visit the OpenAI website
  2. Click on the "Sign Up" button and follow the instructions to create an account.
  3. Once logged in, go to the API keys page.
  4. Click on "Create new secret key" and follow the instrctions. Name the key something meaningful like "QGISGPT API key".
  5. Copy the API key and store it somewhere secure. You will not be able to see the key again after creating it.
  6. Navigate to the Rate limits page and set a modest rate limit. If you intend to only use the OpenAI API key with the QGISGPT Plugin, a rate limit of as little as $5 should suffice. Adjust according to your usage.

Usage

  1. In the Settings tab under API Keys paste your OpenAI API key into the text field.
  2. Paste your assistant ID in the text field just below
  3. In the Chat tab, type your conversation with the assistant in the text area below the main response display
  4. Press enter or click the Chat button to submit. Use Shift+Enter to enter a new line in the chat without submitting.
  5. Wait for the AI to process your request
